== README

This is the Haiku Package Build System (hpbs)

The hpbs instucts worker Haiku systems to build recipies and displays
recipe lint results

=== Important commands

* rake db:reset
  Empty the database
* rake db:migrate
  Create the database
* rake db:seed
  Populate some basic data to get started
* rake recipe:lint
  Performs a lint scan of the current recipes
* rake recipe:sync
  Syncs the applications database of recipes with haikuports
